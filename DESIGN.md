# DESIGN.md

## 项目画像
**全球重要信息简报** - 为投资/研究专业人士提供每日全球要闻聚合 + AI 智能筛选 + 邮件推送。受众：基金经理、研究员、宏观分析师；使用场景：开盘前 8:01 简报查阅、深度分析跳转、邮件订阅。

## 气质与意象
**意象锚点**：1970 年代彭博终端的琥珀色磷光屏 × 顶级财经报纸的版头 — 一边是密不透风的数据流，一边是严谨的版面编排。把"交易大厅的紧张感"和"编辑部的克制感"拼在一起。

**关键词**：终端感、编辑部感、磷光辉、克制的密度、值得信赖。

## 视觉策略
- **图像方向**：纯数据可视化驱动的界面，**不依赖**任何摄影/插图，让信息本身成为主角。
- **图形语言**：细线分隔（1px 边框）、等宽数据列、细颗粒的网格化卡片、状态色块（置信度高/中/低）。
- **图标语言**：Lucide 线性图标为主，stroke-width 1.5，保持克制。

## 配色方案
| 角色 | 颜色 | 意象来源 |
|------|------|---------|
| 背景主色 | `#0A0E1A` 深蓝黑 | 彭博终端的暗色背景 |
| 面板底色 | `#111726` | 终端面板的深灰蓝 |
| 边框线 | `#1E2638` | 几乎不可见的细线 |
| 主文字 | `#E8EAF0` 暖白 | 磷光屏的米白 |
| 次文字 | `#7B8499` 蓝灰 | 屏幕的次级信息 |
| 强调金 | `#D4A24C` 琥珀金 | 磷光屏的标志性暖光 |
| 上涨/利好 | `#3DA37A` 墨绿 | 传统金融绿 |
| 下跌/利空 | `#D45C5C` 砖红 | 传统金融红 |
| 置信度高 | `#3DA37A` | 同上 |
| 置信度中 | `#D4A24C` | 同上 |
| 置信度低 | `#7B8499` | 同上 |

## 字体排版
- **标题/正文中文**：思源宋体（Noto Serif SC）— 编辑部感、严谨
- **数据/时间戳/代码**：JetBrains Mono — 终端感
- **UI 标签/导航**：Inter — 干净中性
- 排版节奏：标题大但克制（2xl/3xl），正文 14-15px，密而不挤。日期、时间戳、置信度一律等宽字体强化"数据"感。

## 动效与交互
- 运动性格：克制、精准、无情绪。`ease-out`，时长 150-220ms。
- 卡片悬停：边框从 `#1E2638` 渐变到 `#D4A24C`（琥珀金描边），无位移。
- 板块切换：内容区淡入淡出，180ms。
- 「详细分析」展开：手风琴式展开，详细分析区带左侧琥珀金竖线标识"深度阅读区"。
- 数字/置信度：进入视口时一次性淡入，不做滚动计数。

## 页面结构
- **首页（简报）**：顶部 sticky 导航条（左：Logo + 日期选择器；右：媒体库 / 模型逻辑 / 邮件设置入口）。主区域分三段：①远期发展（2 条），②国内市场影响（3 条），③未来一周事件（5 条）。每条卡片包含序号、板块标签、正文、出处、置信度徽章、详细分析按钮。
- **媒体库管理**：左侧媒体类型筛选，右侧表格 + 行内编辑 + 增删 + Excel 导入/导出按钮。
- **模型逻辑管理**：左侧参数分类（关键词 / 主题 / 权重 / 排除词），右侧表单 + 版本历史时间线 + "AI 一键优化"按钮。
- **简报信息表管理**：表格 + 增删改 + Excel 导入/导出。
- **未来事件表管理**：表格 + 增删改 + Excel 导入/导出。
- **邮件设置**：收件人列表（多邮箱）、启用开关、发送时间、预览上次发送内容。

## 组件规范
- **置信度徽章**：小尺寸、纯色块 + 文字，无圆角（`rounded-sm`）。高=墨绿底白字；中=琥珀底深色字；低=灰底白字。
- **数据卡片**：`bg-[#111726]` + `border border-[#1E2638]` + `rounded-md`，hover 改 border 颜色。
- **板块标题**：左侧 4px 琥珀金竖线 + 板块名 + 序号范围（如"01 — 02"）。
- **按钮**：主按钮琥珀金底深色字；次按钮透明底琥珀金描边；危险按钮砖红描边。
- **输入控件**：深色底 `#0A0E1A` + 细边框，focus 时琥珀金边框。

## 设计禁忌
- ❌ 不用渐变色块（金融感被破坏）
- ❌ 不用圆角超过 8px 的卡片（保持"工具感"而非"消费感"）
- ❌ 不用大留白 / 居中 hero 区（这是工作台不是营销页）
- ❌ 不用 emoji 和装饰性图标
- ❌ 不用紫色 / 蓝紫渐变（科技蓝紫泛滥色）
- ❌ 不用动画弹跳 / 旋转等活泼动效
- ❌ 数字 / 时间戳 / 置信度不使用衬线字体
